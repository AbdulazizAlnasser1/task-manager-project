# HimmahBoard system 

This repository is created for the Software Engineering course project.

## Project Description
A simple Java console-based Task Manager system that allows users to:
- Add tasks
- View tasks
- Update tasks
- Delete tasks
- Change task status

## Project Scope
This repository contains the project structure and design artifacts only.
The implementation is provided as part of the project documentation and diagrams.

## Technologies
- Java
- UML Diagrams
- Agile (Scrum)
- Jira
- Confluence
